# swe30003

How to run this project:

- Backend (APIs): Install `uv`, then use `uv install` on first run to install packages. Afterwards, use `uv run fastapi dev` while inside the `backend` folder. While it is running, access API menu via `localhost:8000/docs`
- Frontend: Install `npm`, then run `npm install` to install packages, then `npm run dev` to run the frontend interface. Access the interface on `localhost:5173`
